# Task Management App

## Overview
This is a simple Task Management App that allows users to add, view, and manage tasks. Each task includes a **name**, **category**, **deadline**, and **status**. The app displays tasks in a table and highlights their status using colors.

## Features
- **Add Tasks:** Users can add new tasks with a name, category, deadline, and initial status.
- **Update Status:** Tasks can be updated to reflect progress, such as “In Progress,” “Completed,” or “Overdue.”
- **Delete Tasks:** Users can remove tasks from the list.
- **Status Colors:** Tasks are color-coded for easy identification:
  - In Progress → Yellow
  - Completed → Green
  - Overdue → Red
- **Filter Tasks :** Users can filter tasks by status to view specific types

## How to Use
1. Enter the task name, select a category, choose a deadline, and select a status.  
2. Click **Add Task** to add it to the list.  
3. The task appears in the table below with the status color applied.  
4. Use the **Delete** button to remove tasks as needed.  
5. (Optional) Use the status filter buttons to show tasks with a specific status.
